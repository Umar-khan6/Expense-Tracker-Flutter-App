A modern **Expense Tracker Mobile App** built with **Flutter & Firebase** to help users manage their daily expenses efficiently.

The app provides **real-time expense tracking**, **financial analytics**, **category management**, and **secure authentication** with a clean **Material Design 3 UI**.

---

## 🚀 Features

- **User Authentication**
  - Firebase Email & Password Authentication
  - Secure login & signup system

- **Real-Time Expense Tracking**
  - Add, edit, and delete expenses
  - Real-time Firestore database updates

- **Category Management**
  - Create and organize expense categories
  - Filter expenses by category

- **Financial Dashboard**
  - Total spending overview
  - Expense summaries & analytics
  - Visual financial insights

- **Cross-Platform Support**
  - Android support
  - iOS support
  - Responsive Flutter UI

- **Modern UI**
  - Material Design 3
  - Smooth animations & clean interface

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Umar-khan6/expense-tracker-app.git

cd expense-tracker-app
```

### 2. Install Dependencies

```bash
flutter pub get
```

### 3. Run the App

```bash
flutter run
```
